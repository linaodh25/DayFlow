# DayFlow

**Smart Daily Planner & Habit Tracker** — Manage tasks, build habits, track progress.

![Flutter](https://img.shields.io/badge/Flutter-02569B?style=flat&logo=flutter&logoColor=white)
![Firebase](https://img.shields.io/badge/Firebase-FFCA28?style=flat&logo=firebase&logoColor=black)

---

## 🚀 Features

- ✅ Tasks with priorities, tags & subtasks
- 🔥 Habit streaks & completion tracking
- 📝 Smart notes with pinning & search
- ⏰ Custom reminders & recurring alerts
- ☁️ Real-time cloud sync (Firebase)
- 🎨 Dark/Light mode + EN/FR/AR support
- 📊 Productivity analytics

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|
| Framework | Flutter (Dart) |
| Backend | Firebase (Auth, Firestore, Storage) |
| State | Bloc/Cubit |
| Analytics | Mixpanel |
